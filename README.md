## 1. Deskripsi Singkat
Pada praktikum ini, saya mempelajari dasar-dasar pengembangan layanan web (Web Service) menggunakan **Node.js** dan framework **Express.js**. Fokus utama praktikum ini adalah memahami struktur proyek, routing dasar, dan penanganan request/response sederhana.

## 2. Fitur / Materi yang Dipelajari
* Inisialisasi proyek Node.js (`package.json`).
* Instalasi dan penggunaan **Express.js**.
* Pembuatan Route dasar (GET, POST).
* Penggunaan **Nodemon** untuk development (opsional/jika ada).
* Struktur folder dasar aplikasi backend.

## 3. Prasyarat & Instalasi
Pastikan **Node.js** sudah terinstall di komputer.

1.  Clone atau download repositori ini.
2.  Buka terminal di folder proyek.
3.  Install dependencies:
    ```bash
    npm install
    ```

## 4. Cara Menjalankan Server
Untuk menjalankan server dalam mode development:

```bash
node index.js
```
Atau jika menggunakan script start:
```bash
npm start
```
Jika berhasil, akan muncul pesan di terminal:
```
Server running on http://localhost:3000
```
